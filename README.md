# Multi-Tier Web Application Deployment Project

## Project Overview
This project demonstrates the deployment of a multi-tier web application using **Terraform** for infrastructure provisioning and **Ansible** for configuration management. The application includes:

- **Load-balanced WordPress web servers**
- **MySQL database server running in Docker**
- **Node.js Todo application**
- **Nginx load balancer**

The project is designed to simulate a real-world multi-tier environment with automated provisioning and configuration.

## Technologies Used
- Terraform
- Ansible
- Docker
- Nginx
- Apache, PHP, MySQL
- Node.js

## Key Features
- Infrastructure as Code using Terraform
- Automated server configuration using Ansible
- Docker-based database deployment
- Load-balanced WordPress deployment
- Node.js application deployment
